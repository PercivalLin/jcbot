# Security Policy

Lobster can control a real desktop environment, so safety and secret handling matter.

## Report a vulnerability

Do not open a public issue for a security problem. Report it privately by email:

- `Lin18974@outlook.com`

Include:

- what you found
- how it can be reproduced
- whether any secrets, tokens, or local files may be exposed
- your preferred contact method

## What to avoid in public issues

- API keys
- bot tokens
- local file paths that contain sensitive data
- screenshots that expose private content

## Scope

This repository is public, but runtime credentials, certs, and secret files are expected to stay local and out of Git history.
