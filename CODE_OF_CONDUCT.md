# Code of Conduct

We want Lobster to be a practical project that people can use, test, and improve without friction.

## Expected behavior

- Be direct and respectful.
- Focus on the code, docs, and user impact.
- Give actionable feedback.
- Avoid harassment, insults, and gatekeeping.

## Unacceptable behavior

- Personal attacks
- Doxxing or exposing private data
- Deliberately sabotaging other contributors' work
- Misusing the project to bypass privacy, safety, or approval boundaries

## Enforcement

Maintainers may close, edit, or remove content that violates this code of conduct and may restrict contribution access if needed.
